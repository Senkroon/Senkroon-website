# Repository Instructions for Agents

This repository is a Laravel web application with a Porto theme frontend.

Use `AGENTS.md` as the primary project guide. It contains the main technology stack, route definitions, key files, and agent workflow.

For quick reference:
- Blade templates: `resources/views`
- Main layout: `resources/views/layout/master.blade.php`
- CSS overrides: `public/porto/css/custom.css` + `public/porto/css/how_we_work.css`
- Custom JS: `public/porto/js/custom.js`
- Routes: `routes/web.php`

Do not modify `vendor/` or theme vendor files unless explicitly requested.
