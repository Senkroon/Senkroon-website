# Copilot Project Instructions

This project is a Laravel Blade application using a Porto theme frontend.

Use `AGENTS.md` as the main guide for technology, file structure, route names, and project conventions.

Important points:
- Blade templates are in `resources/views`
- Main layout is `resources/views/layout/master.blade.php`
- CSS overrides are in `public/porto/css/custom.css` and `public/porto/css/how_we_work.css`
- JavaScript customizations are in `public/porto/js/custom.js`
- Preserve Turkish text unless the user requests otherwise
- Avoid modifying `vendor/` or compiled theme vendor files unless necessary
